# 이럴 때 사용해요

- 새로운 서비스를 출시하거나 구상할 때 
- 서비스 내에서 새로운 기능을 개발할 때
- 경쟁사를 기준으로 목표를 수립할 때

# 이렇게 해보세요

1. 벤치마킹에 필요한 항목을 반영해 양식을 만든다.
2. 해당 분야의 뉴스나 브랜드 사이트 서칭하고 타 서비스를 탐색한다.
3. 실제 서비스를 사용해보고 양식을 기록한다.
4. 수집한 자료를 바탕으로 결론을 도출한다.
