# 이럴 때 사용해요

- 정량적 데이터만으로는 파악하기 어려운 사용자 행동의 이유와 맥락을 파악하고 싶을 때

# 이렇게 해보세요

1. 인터뷰 목적을 설정한다.
2. 적합한 인터뷰 방식을 선택한다.
3. 인터뷰 대상을 선정하고 참가자를 모집한다.
4. 인터뷰 질문지를 작성한다.
5. 인터뷰를 진행한다.
6. 인터뷰 내용을 정리하고 인사이트를 도출한다.

# 인터뷰 방법
- 심층 인터뷰, 맥락적 인터뷰, 포커스 그룹 인터뷰​​​​​​​
